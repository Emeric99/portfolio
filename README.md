# Portfolio – Emeric Tcholagheu

Mein persönliches Portfolio, entwickelt mit [Next.js](https://nextjs.org) und bereitgestellt auf [Vercel](https://vercel.com).

## Lokale Ausführung

Zuerst den Entwicklungsserver starten:

```bash
npm run dev
```

Anschließend im Browser öffnen: [http://localhost:3000](http://localhost:3000)

## Tech-Stack

- **Next.js** – React-Framework für modernes Webentwicklung
- **TypeScript** – Typsicheres JavaScript
- **Tailwind CSS** – Utility-first CSS Framework
- **Vercel** – Deployment-Plattform

## Deployment

Das Portfolio ist live verfügbar auf Vercel.  
 [Portfolio ansehen]((https://portfolio-pearl-six-98.vercel.app))
